# chatbotAI

Kho lưu trữ cho các chatbot nội bộ và bộ tài liệu liên quan.

## Cấu trúc hiện tại

- `chatbots/HR_OD/` - không gian dành cho chatbot HR/OD
- `shared/` - tài nguyên dùng chung cho nhiều chatbot
- `archives/` - tài liệu cũ hoặc bản lưu trữ

## Chatbot hiện có

- HR/OD Advisor

## Ghi chú

Các tài liệu gốc hiện vẫn đang ở thư mục gốc để giữ nguyên cấu trúc cũ. Khi sẵn sàng, có thể chuyển dần vào `chatbots/HR_OD/`.
