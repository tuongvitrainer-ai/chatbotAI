# HR/OD Chatbot

Thư mục dành riêng cho chatbot HR/OD.

## Gợi ý tổ chức

- `source/` - tài liệu nguồn, brief, khung nghiệp vụ
- `knowledge/` - hệ thống kiến thức, playbook, framework
- `prompts/` - prompt, instruction, router
- `tests/` - bộ test, scenario, rubric
- `outputs/` - bản xuất, báo cáo, kết quả đánh giá
