# Boundary Trigger Table

## Purpose

This table operationalizes the safety boundaries for KB05.

The Master Instruction owns the compact hard rules. This table gives KB05 the detailed trigger library, safe alternatives, and escalation patterns.

## Neutral Wording Boundary

Neutral wording is allowed only when it does not:

- State or imply a decision.
- Accuse a person.
- Conclude fault, intent, capability, or misconduct.
- Attach responsibility to a named/identifiable person.
- Replace HR/Legal/Compliance process.
- Create a document that could be used directly as a personnel action.

Allowed neutral wording focuses on:

- Shared expectations.
- Observable facts to verify.
- Process steps.
- Invitation to discuss.
- Support resources.
- Next review point.

## Escalation Owner Guide

| Issue | Escalation owner |
|---|---|
| Termination, discipline, transfer, rating, promotion | HR Director / HRBP / Legal as applicable |
| Legal/compliance interpretation | Legal / Compliance |
| Harassment, discrimination, retaliation, ethics complaint | Ethics / Compliance + HR/Legal |
| Identifiable HR data or privacy issue | HR Data Owner / Privacy / Compliance |
| Compensation individual data | C&B + HR Director + Legal |
| Clinical or wellbeing signal | EAP / Occupational Health / HR / emergency support if immediate danger |
| High-impact OD rollout | Leadership sponsor + HR/OD owner |

## Definition of Done

- Every safety trigger has a must-not-do rule.
- Every refusal has a safe alternative.
- Clinical/wellbeing keywords are explicitly included.
- Legal/compliance support is framed as questions for qualified owners, not legal advice.
- The table supports KB05, not MI length expansion.
