### Out-of-scope note — không mở rộng keyword

Các miền sau **không mở rộng keyword ở giai đoạn này** vì dễ làm chatbot lệch scope hoặc tạo rủi ro: legal advice/labor law deep dive, termination/discipline/transfer decision, individual performance scoring, personality judgment, sensitive employee data processing, employee relations investigation, compensation & benefits design chi tiết, payroll/HR admin, recruitment sourcing/interview scoring, clinical/mental health coaching, business strategy consulting tổng thể.

Lý do loại: các miền này không phục vụ trực tiếp KB01–KB04 ở vai trò OD Advisor, hoặc chỉ nên xuất hiện như **boundary note / escalation trigger / safe alternative artifact**.

## Gợi ý dùng cho Facet Matrix

Từ bộ keyword Core còn lại, nên tạo Facet Matrix theo 7 facet:

1. **Domain facet:** OD Foundation / Enterprise Context / Org Design / Culture / Engagement / Performance / Capability / Leadership / Change.
2. **System facet:** Structure / Role / Process / Governance / Culture / Leadership / Measurement.
3. **Stakeholder facet:** Corporate center / BU / HR Manager / OD Manager / HRBP / Leadership / Line Manager / Employee.
4. **Problem signal facet:** Silo / Role overlap / Low trust / Low engagement / KPI misalignment / Manager inconsistency / Resistance.
5. **Intervention bridge facet:** Clarify / Align / Redesign / Enable / Reinforce / Measure / Improve.
6. **Artifact facet:** Concept note / Diagnostic note / Roadmap / Manager guide / Employee FAQ / Executive memo / Dashboard outline.
7. **Boundary facet:** Legal / Individual judgment / Sensitive data / HR ops / C&B / Recruitment / Business strategy.
