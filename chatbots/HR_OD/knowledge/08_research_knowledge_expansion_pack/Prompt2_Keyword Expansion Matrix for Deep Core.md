## Gợi ý dùng cho bước tiếp theo: Facet Matrix

Nên chuyển Keyword Matrix này thành **Facet Matrix** theo 6 facet:

1. **Problem facet:** Culture / Performance / Capability / Org Design / Change / Engagement / Leadership.
2. **Layer facet:** Individual / Team / Manager / Process / Structure / Culture / Leadership / System.
3. **Evidence facet:** Survey / Interview / HR metrics / Business signal / Observation / Feedback.
4. **Intervention facet:** Diagnostic / Org Design / Culture / Leadership / Manager / Performance / Capability / Change / Engagement / Measurement.
5. **Artifact facet:** Framing / Diagnostic / Roadmap / Memo / Guide / Dashboard / Safe Alternative.
6. **Risk facet:** Missing data / Overclaim / Legal / Individual judgment / Sensitive data / Scope drift.
